Lightweight floating chat button plugin for WordPress with customizable UI options.

== Description ==

Instant Chat Connect allows you to add a customizable floating chat button to your WordPress website.

The plugin is lightweight, performance-focused, and does not collect or transmit any user data.

Features:

* Custom logo selection (standard or business style)
* Icon size control
* Optional text bubble support
* Left / Right positioning
* Clean OOP architecture
* No external libraries or tracking scripts
* Fully GPL compatible

This plugin does not integrate with external APIs in the free version and does not send any data to third-party servers.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/instant-chat-connect` directory.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Configure the plugin from the admin menu.

== Frequently Asked Questions ==

= Does this plugin collect user data? =
No. The plugin does not collect, store, or transmit user data.

= Does it require external services? =
No. The plugin works independently without external API connections.

= Is it lightweight? =
Yes. It does not load external libraries or remote scripts.
